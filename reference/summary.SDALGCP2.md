# Summary of an SDALGCP2 fit

Summary of an SDALGCP2 fit

## Usage

``` r
# S3 method for class 'SDALGCP2'
summary(object, ...)
```

## Arguments

- object:

  an object of class `"SDALGCP2"` from
  [`mcml_fit`](https://olatunjijohnson.github.io/SDALGCP2/reference/mcml_fit.md).

- ...:

  unused.

## Value

an object of class `"summary.SDALGCP2"` with a coefficient table.
