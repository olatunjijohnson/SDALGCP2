# Plot an SDALGCP2 fit (the phi profile deviance)

Plot an SDALGCP2 fit (the phi profile deviance)

## Usage

``` r
# S3 method for class 'SDALGCP2'
plot(x, ...)
```

## Arguments

- x:

  an `"SDALGCP2"` object.

- ...:

  passed to
  [`phi_profile`](https://olatunjijohnson.github.io/SDALGCP2/reference/phi_profile.md).

## Value

invisibly, the profile (see
[`phi_profile`](https://olatunjijohnson.github.io/SDALGCP2/reference/phi_profile.md)).
