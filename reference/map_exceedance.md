# Map exceedance probabilities P(relative risk \> threshold)

Map exceedance probabilities P(relative risk \> threshold)

## Usage

``` r
map_exceedance(x, threshold = 1, bound = NULL, ...)
```

## Arguments

- x:

  an `"SDALGCP2_pred"` object.

- threshold:

  a single relative-risk threshold.

- bound:

  optional `sf` boundary (continuous only).

- ...:

  unused.

## Value

a `ggplot` object.
